## Prerequisites
- Anaconda

## Setup
```bash
conda create -n ai_env python=3.10
conda activate ai_env
pip install -r requirements.txt
```

## Usage

```bash
# clone .env file
cp -r .env.example .env

# modify parameters in .env file

# start app
streamlit run Welcome.py
```
